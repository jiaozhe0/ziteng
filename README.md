## Html5与app共用页面开发
> h5页面的路径参数:html  原生app的路径参数:app

#### 协议静态app页面：
* [url]/#/agreement/user/app : 用户协议
* [url]/#/agreement/service/app : 用户服务协议
* [url]/#/agreement/server/app : 服务者管理协议

* [url]/#/agreement/user/html : 用户协议
* [url]/#/agreement/service/html : 用户服务协议
